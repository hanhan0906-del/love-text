# 🌟 新手操作指南 - 零基础也能看懂

## 第一步：测试你的网站

### 🎯 最简单的方法（推荐新手）

1. **找到 `index.html` 文件**
   - 就在你的文件夹里
   - 图标是浏览器的图标

2. **双击打开它**
   - 会自动用浏览器打开
   - 如果没打开，右键 → 打开方式 → 选择浏览器

3. **输入激活码测试**
   ```
   LOVE2024-001
   ```
   - 复制这个激活码
   - 粘贴到输入框
   - 点击"开始测试"

4. **完成测试**
   - 回答15道题
   - 查看你的结果

## 第二步：修改激活码

### 📝 如果你想改成自己的激活码

1. **找到 `codes.js` 文件**
   - 右键 → 打开方式 → 记事本（或其他文本编辑器）

2. **找到这一段**（在文件开头）
   ```javascript
   const ACTIVATION_CODES = [
       "LOVE2024-001", "LOVE2024-002", "LOVE2024-003",
       // ... 更多激活码
   ];
   ```

3. **修改激活码**
   - 把 `LOVE2024-001` 改成你想要的，比如 `MYCODE-001`
   - 注意：每个激活码要用**英文双引号**包起来
   - 激活码之间用**英文逗号**分隔
   - 最后一个激活码后面**不要**加逗号

4. **保存文件**
   - 点击 文件 → 保存
   - 或按 `Ctrl + S`

5. **刷新浏览器测试**
   - 回到浏览器
   - 按 `F5` 刷新
   - 或按 `Ctrl + F5` 强制刷新
   - 输入你的新激活码测试

### ⚠️ 常见错误

**错误示例 1：忘记引号**
```javascript
❌ 错误：LOVE2024-001
✅ 正确："LOVE2024-001"
```

**错误示例 2：用了中文符号**
```javascript
❌ 错误："LOVE2024-001"，"LOVE2024-002"  （中文逗号）
✅ 正确："LOVE2024-001", "LOVE2024-002"  （英文逗号）
```

**错误示例 3：最后多了逗号**
```javascript
❌ 错误：
[
    "LOVE2024-001",
    "LOVE2024-002",  ← 这里不该有逗号
]

✅ 正确：
[
    "LOVE2024-001",
    "LOVE2024-002"   ← 最后一个不加逗号
]
```

## 第五步：实现实时预览（强烈推荐！）

### 💻 使用 VS Code（免费软件）

1. **下载 VS Code**
   - 访问：https://code.visualstudio.com/
   - 点击下载 Windows 版本
   - 安装（一直点下一步）

2. **打开你的项目**
   - 启动 VS Code
   - 点击 文件 → 打开文件夹
   - 选择你的"恋爱理想型测试"文件夹

3. **安装 Live Server 插件**
   - 点击左侧的"扩展"图标（四个方块）
   - 搜索 "Live Server"
   - 点击 "安装"

4. **启动实时预览**
   - 在左侧文件列表中，右键点击 `index.html`
   - 选择 "Open with Live Server"
   - 浏览器会自动打开网站

5. **享受实时预览**
   - 修改任何文件（HTML、CSS、JS）
   - 保存后浏览器**自动刷新**
   - 立即看到效果！

### 🎉 现在你可以：
- 修改代码
- 按 `Ctrl + S` 保存
- 浏览器自动刷新
- 立即看到效果
- 不用手动刷新！

## 第七步：分享给客户

### 📦 方法一：打包发送（适合小规模）

1. **选中所有文件**
   - `index.html`
   - `style.css`
   - `app.js`
   - `codes.js`
   - `questions.js`

2. **右键 → 发送到 → 压缩文件夹**

3. **发送给客户**
   - 通过微信、QQ等发送
   - 客户解压后双击 `index.html` 即可使用

### 🌐 方法二：上传到网站（适合大规模）

**使用 GitHub Pages（完全免费）**

1. **注册 GitHub 账号**
   - 访问：https://github.com
   - 点击 Sign up（注册）

2. **创建仓库**
   - 点击右上角 "+" → "New repository"
   - 名字随便起，比如 "love-test"
   - 选择 "Public"（公开）
   - 点击 "Create repository"

3. **上传文件**
   - 点击 "uploading an existing file"
   - 把你的5个文件拖进去
   - 点击 "Commit changes"

4. **开启 GitHub Pages**
   - 点击 "Settings"（设置）
   - 左侧找到 "Pages"
   - Source 选择 "main"
   - 点击 "Save"

5. **获取网址**
   - 等待1-2分钟
   - 会显示一个网址，类似：
     ```
     https://你的用户名.github.io/love-test/
     ```
   - 这就是你的网站地址！

6. **分享给客户**
   - 直接发送网址
   - 客户打开就能用
   - 手机电脑都能访问

## 🆘 遇到问题怎么办？

### 问题1：网页打开是乱码
- **原因**：文件编码问题
- **解决**：
  1. 用 VS Code 打开文件
  2. 右下角点击编码（比如显示 GBK）
  3. 选择 "Save with Encoding"
  4. 选择 "UTF-8"

### 问题2：修改后没效果
- **解决**：
  1. 检查是否保存了文件
  2. 按 `Ctrl + F5` 强制刷新浏览器
  3. 清除浏览器缓存

### 问题3：网页显示错误
- **解决**：
  1. 按 `F12` 打开开发者工具
  2. 点击 "Console"（控制台）
  3. 看看有什么红色错误信息
  4. 通常是代码写错了（比如少了引号、逗号）

### 问题4：激活码用完了
- **解决**：
  1. 打开 `codes.js`
  2. 在激活码列表最后添加新的
  3. 或者清除浏览器 localStorage（见使用说明）
