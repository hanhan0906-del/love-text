# 🧪 快速测试指南

## 立即测试你的网站

### 方法一：最快速（推荐）

1. **双击打开 `index.html`**
   - 文件会在浏览器中打开

2. **输入测试激活码**
   ```
   LOVE2024-001
   ```

3. **开始答题**
   - 随便选择答案
   - 看看能否正常进入下一题

4. **查看结果**
   - 答完15题后会显示结果
   - 检查结果是否正常显示

### 方法二：使用本地服务器

1. **打开 PowerShell**
   - 在文件夹空白处，按住 `Shift` 键
   - 右键点击，选择"在此处打开 PowerShell 窗口"

2. **启动服务器**
   ```powershell
   python -m http.server 8000
   ```
   
   如果提示找不到 Python，试试这个：
   ```powershell
   python3 -m http.server 8000
   ```

3. **打开浏览器**
   - 访问：`http://localhost:8000`

4. **测试功能**
   - 输入激活码：`LOVE2024-001`
   - 完成测试

## 常见问题排查

### 问题1：双击 index.html 打不开

**可能原因**：没有默认浏览器

**解决方法**：
1. 右键点击 `index.html`
2. 选择"打开方式"
3. 选择浏览器（Chrome、Edge、Firefox等）

---

### 问题2：页面显示乱码

**可能原因**：文件编码问题

**解决方法**：
1. 用 VS Code 打开 `index.html`
2. 右下角点击编码
3. 选择"Save with Encoding"
4. 选择"UTF-8"
5. 保存文件

---

### 问题3：激活码验证不工作

**排查步骤**：

1. **打开开发者工具**
   - 按 `F12`
   - 点击"Console"（控制台）标签

2. **查看错误信息**
   - 如果有红色错误，截图或记录下来

3. **检查文件加载**
   - 点击"Network"（网络）标签
   - 刷新页面
   - 检查 `codes.js`、`app.js`、`questions.js` 是否成功加载

4. **检查文件路径**
   - 确保所有文件在同一文件夹
   - 文件名必须完全一致（包括大小写）

---

### 问题4：修改代码后没效果

**解决方法**：

1. **强制刷新浏览器**
   - 按 `Ctrl + F5`
   - 或 `Ctrl + Shift + R`

2. **清除缓存**
   - 按 `F12` 打开开发者工具
   - 右键点击刷新按钮
   - 选择"清空缓存并硬性重新加载"

3. **检查文件是否保存**
   - 确认修改的文件已保存
   - VS Code 中，文件名旁边不应该有圆点

---

### 问题5：激活码用完了怎么办

**方法一：清除已使用记录**
1. 按 `F12` 打开开发者工具
2. 点击"Console"（控制台）
3. 输入：`localStorage.clear()`
4. 按回车
5. 刷新页面

**方法二：添加新激活码**
1. 打开 `codes.js`
2. 在激活码列表末尾添加新的
3. 保存文件
4. 刷新浏览器
