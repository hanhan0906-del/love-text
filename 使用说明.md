# 恋爱理想型测试网站 - 使用说明

## 🎯 项目概述

这是一个完全在浏览器中运行的测试网站，不需要服务器后端，所有功能都通过 HTML + CSS + JavaScript 实现。

## 📂 文件结构

```
恋爱理想型测试/
├── index.html          # 网页主文件
├── style.css           # 样式文件
├── app.js             # 主程序逻辑
├── codes.js           # 激活码管理
├── questions.js       # 题目和结果数据
└── 使用说明.md        # 本文件
```

## 🚀 如何使用

### 方法一：直接双击打开（最简单）
1. 找到 `index.html` 文件
2. 双击打开（会用默认浏览器打开）
3. 输入激活码：`LOVE2024-001`（或其他001-500的任意编号）
4. 开始测试

### 方法二：使用本地服务器（推荐）
1. 按住 `Shift` 键，右键点击文件夹空白处
2. 选择"在此处打开 PowerShell 窗口"
3. 输入命令：`python -m http.server 8000`
4. 打开浏览器，访问：`http://localhost:8000`

## 🔑 激活码说明

### 当前激活码
- 格式：`LOVE2024-001` 到 `LOVE2024-500`
- 总数：500个
- 特点：每个激活码只能使用一次

### 如何修改激活码

1. 打开 `codes.js` 文件
2. 找到 `ACTIVATION_CODES` 数组
3. 修改或添加激活码，例如：

```javascript
const ACTIVATION_CODES = [
    "LOVE2024-001",
    "LOVE2024-002",
    "你的新激活码",
    // ... 更多激活码
];
```

### 如何重置已使用的激活码

**方法一：清除浏览器数据**
1. 按 `F12` 打开开发者工具
2. 点击 `Application`（应用程序）标签
3. 左侧找到 `Local Storage`
4. 删除 `usedCodes` 项

**方法二：在控制台清除**
1. 按 `F12` 打开开发者工具
2. 点击 `Console`（控制台）标签
3. 输入：`localStorage.clear()`
4. 按回车

## 📊 测试题目说明

### 当前配置
- 题目数量：15题
- 选项类型：每题4个选项（A/B/C/D）
- 结果类型：6种（浪漫主义型、理性务实型、自由独立型、温柔体贴型、激情冒险型、平衡和谐型）

### 如何修改题目

打开 `questions.js` 文件，找到 `QUESTIONS` 数组：

```javascript
{
    id: 1,
    question: "你的问题内容？",
    options: [
        { text: "选项A", value: "A" },
        { text: "选项B", value: "B" },
        { text: "选项C", value: "C" },
        { text: "选项D", value: "D" }
    ]
}
```

### 如何修改结果

在 `questions.js` 文件中找到 `RESULT_TYPES`：

```javascript
"你的结果类型名称": {
    description: "详细描述...",
    traits: ["特征1", "特征2", "特征3"],
    advice: [
        "建议1",
        "建议2",
        "建议3"
    ]
}
```

## 🔧 常见问题

### 1. 激活码输入后提示"已被使用"
- 原因：浏览器记录了使用历史
- 解决：清除浏览器 localStorage（见上方"如何重置已使用的激活码"）

### 2. 修改代码后网页没变化
- 解决：按 `Ctrl + F5` 强制刷新浏览器

### 3. 想要实时预览修改效果
- 推荐使用 VS Code 编辑器
- 安装 "Live Server" 插件
- 右键 `index.html` → "Open with Live Server"
- 修改代码后自动刷新浏览器

### 4. 如何分享给别人使用

**方法一：打包发送**
- 将所有文件打包成 zip
- 发送给对方
- 对方解压后双击 `index.html` 即可使用

**方法二：上传到网站**
- 使用 GitHub Pages（免费）
- 使用 Netlify（免费）
- 使用 Vercel（免费）
